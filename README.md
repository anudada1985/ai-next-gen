# AI NEXTGEN - Deploy Ready (JavaScript)

This is a ready-to-deploy Next.js (App Router) + TailwindCSS starter for AI NEXTGEN.
Theme: Calm & Modern (white + blue)
Tagline: Empowering Students with Intelligent Skills

## Quick start (after unzip):
1. cd ai-nextgen
2. npm install
3. npm run dev
4. Open http://localhost:3000

## Deploy to Vercel
1. Push to GitHub repository
2. Import repo on Vercel and deploy
3. Add domain ainextgen.com.pk in Vercel and ensure DNS A + CNAME are set
